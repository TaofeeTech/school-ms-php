# School Management System

School Management System is a comprehensive web application developed using **Core PHP**, **MySQL**, and **Functional PHP**. The project is designed to simulate a real-world school administration system while strengthening backend development skills without relying on frameworks.

The application focuses on **clean architecture**, **reusable code**, **database design**, **authentication**, **authorization**, and **modern software engineering practices**. It serves as both a learning project and a portfolio piece, demonstrating how scalable, secure, and maintainable applications can be built using **vanilla PHP**.

## Technologies

- PHP 8+
- MySQL
- HTML5
- CSS3
- JavaScript
- Composer
- PHPUnit
- Git & GitHub

## Project Goals

- Simulate a real-world school administration system.
- Strengthen backend development skills without using a PHP framework.
- Follow modern coding standards and best practices.
- Create a secure and maintainable codebase.
- Demonstrate professional software development workflows.
